Todo web app application built in ReactJs from scratch for learning purposes.

### `yarn start`
To start dev server.

### Start json server
*Install json server*
`npm install -g json-server`
*Start json server*
`json-server -p 8080 --watch db.json`
*Test json server is running*
`curl http://localhost:8080/todos`
